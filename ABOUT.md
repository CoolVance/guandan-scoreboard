# About Guandan Scoreboard / 关于本项目

## English

**Guandan Scoreboard** was created with a simple goal: to make scorekeeping for card games effortless and error-free. Whether you are playing a casual game with friends or a competitive match, this tool helps you focus on the game, not the math.

This project is also a demonstration of **AI-Assisted Development**. It was built through a collaborative effort between a human developer and an advanced AI coding agent. Every feature, from the draggable menu to the smart lock logic, was iteratively refined to ensure the best user experience.

We believe in open source and community collaboration. We hope this tool serves you well!
